# AharaSetu - Pre-Download Checklist ✅

Before you download the ZIP, make sure you have everything ready:

## System Requirements

- [ ] Node.js 18+ installed
- [ ] pnpm package manager installed
- [ ] ~500MB free disk space
- [ ] Access to Neon (neon.tech) for database

## Pre-Download Setup

1. [ ] Create a Neon account at https://neon.tech (free tier is fine)
2. [ ] Create a new Neon project
3. [ ] Copy your connection string (save it somewhere safe)

## Files You'll Need

After download, you should have:
- [ ] All source code in the `app/`, `components/`, `lib/` folders
- [ ] Database scripts in `scripts/` folder (`001-create-tables.sql`, `002-seed-data.sql`)
- [ ] This checklist and guides: `QUICKSTART.md`, `SETUP.md`, `TROUBLESHOOTING.md`
